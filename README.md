# Projeto Final DevOps

API CRUD de produtos desenvolvida para a disciplina de DevOps.

## Tecnologias Utilizadas

- Python
- FastAPI
- MySQL
- MongoDB
- Docker
- Docker Compose
- HTML
- CSS
- JavaScript

## Funcionalidades

- Listar produtos
- Criar produtos
- Buscar produtos
- Atualizar produtos
- Remover produtos

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Dyego-Kelven/projeto-final-devops.git
```

Execute os containers:

```bash
docker compose up --build
```

## Swagger da API

Acesse:

```text
http://localhost:8000/docs
```

## Front-end

Abra o arquivo:

```text
frontend/index.html
```

utilizando a extensão Live Server no VS Code.

## Estrutura do Projeto

```text
frontend/
app/
db/
openapi/
scripts/
docker-compose.yml
```

## Objetivo

O projeto tem como objetivo demonstrar a aplicação prática dos conceitos de DevOps, utilizando conteinerização, integração entre front-end e back-end, banco de dados e versionamento com GitHub.
